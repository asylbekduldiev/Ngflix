# ngflix

Steps to launch:

Step-1: 
  `json-server --watch db.json --port 3000`

Step-2:
  `npm install`
  
Step-3:
  to run frontend
  `ng serve`
  
Requirements
	•	Node.js (v18+ recommended)
  • npm
	•	Angular CLI (v20+)
